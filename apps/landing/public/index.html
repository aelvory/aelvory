API testing that travels with you.

Aelvory is an open-source toolbox for hitting HTTP APIs and WebSocket endpoints. Same Vue UI, same encrypted sync, three places to run it: a Tauri desktop app, a VSCode extension, and a self-hostable sync server.

Download View on GitHub

Free. No sign-up to use locally. Sync only when you opt in.

What's in the toolbox

HTTP requests

Methods, headers, bodies (raw, form, GraphQL), pre/post scripts with assertions, response viewer with pretty-printed JSON / XML and search. cURL paste-and-import.

WebSocket testing

Open ws:// / wss:// connections, save a library of named messages, fire them with one click, and read the live log with collapsible bursts. Built for chatty real-time APIs.

Encrypted sync

Argon2id-derived key wraps a per-organization data key with XChaCha20-Poly1305. Your secrets stay encrypted on the server; only your devices have the keys. Last-writer-wins per row.

Collections & environments

Nest collections to any depth, inherit auth and variables from parents, scope variables per environment. Switch envs from a picker — same requests against dev / staging / prod.

Scripts & tests

JavaScript pre-request and post-response hooks. Set environment vars, assert with aelvory.expect, log to the in-app console. A green dot marks tabs that have active code.

Imports & captures

OpenAPI / Swagger (JSON or YAML, file or URL), Postman v2.1, Insomnia exports, HAR captures. Pick a file in your file manager — or right-click in VSCode's Explorer.

Three ways to run it

Desktop app

A Tauri 2 build (Rust + system webview). Native menus, OS file dialogs, no Electron bloat. Around 10 MB installed.

  • Windows · macOS · Linux
  • x64 + arm64 where supported

VSCode extension

The same Vue UI inside a webview tab, with a sidebar showing workspace status and quick actions. SQLite is sql.js (WebAssembly), so a single .vsix runs everywhere.

  • Marketplace: search "Aelvory"
  • Or install from .vsix

Sync server

Optional. ASP.NET Core 10 + Postgres + Redis, distributed via container image. Run it on your own infra; your devices sync through it without exposing plaintext secrets.

  • Docker / Compose
  • SignalR realtime

Open source. Self-hostable. Yours.

Aelvory is built in the open under permissive licensing. No telemetry. The desktop app and the extension run fully offline by default — sync is opt-in, points at whatever server URL you configure, and ships nothing in plaintext.
